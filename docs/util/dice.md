## 概要

サイコロ.

```txt
       _______
      /\      \           N(U)
     /  \   T  \           |            4
    /    \______\  W(L) - T,B - E(R)  2 1 5 6
    \ S  /      /          |            3
     \  /   E  /          S(D)
      \/______/
```

### 注意

もう1種類のサイコロ(3と4が逆)なら, `build`の中をコメントアウトする.

```txt
  3
2 1 5 6
  4
```

## 使用例

* `Dice dc(T, S)` : TとSが上のような位置関係のサイコロを作成.
* `dc.rot(R)` : 右方向に回転.
* `dc.get(idx)` : 側面の値を返す.
  * `idx` : 0から順に`R`, `U`, `L`, `D`.
* `ddx[], ddy[]` : 下を$y$, 右を$x$としたときの進む方向.
